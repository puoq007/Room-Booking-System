🏢 Room Booking System
แอปพลิเคชัน มือถือ ที่พัฒนาด้วย Flutter เพื่อมอบประสบการณ์การจัดการและจองห้องพักที่ทันสมัยและสะดวกสบายให้กับผู้ใช้งาน

📝 เกี่ยวกับโปรเจกต์
Room Booking System ได้รับการออกแบบมาเพื่อแก้ปัญหาความยุ่งยากในการจองห้องประชุม ห้องเรียน หรือพื้นที่ต่าง ๆ ภายในองค์กรหรือสถาบัน ด้วย อินเทอร์เฟซที่ใช้งานง่าย แอปพลิเคชันนี้จะแสดงสถานะห้องว่างแบบเรียลไทม์ ช่วยให้ผู้ใช้สามารถดูข้อมูลห้อง จอง และจัดการการจองได้อย่างมีประสิทธิภาพ

คุณสมบัติหลัก:

ดูห้องว่าง: แสดงรายละเอียดของห้องแต่ละห้อง รวมถึงความจุ สิ่งอำนวยความสะดวก และรูปภาพ

จองห้อง: เลือกวันที่และเวลาที่ต้องการได้อย่างง่ายดาย

จัดการการจอง: ตรวจสอบ แก้ไข หรือยกเลิกการจองที่มีอยู่

แจ้งเตือน: ระบบแจ้งเตือนอัตโนมัติสำหรับการจองที่กำลังจะมาถึง

รองรับหลายแพลตฟอร์ม: แอปพลิเคชันสามารถใช้งานได้ทั้งบน Android และ iOS

ขั้นตอนการติดตั้ง

cd Room-Booking-System

ติดตั้ง dependencies ที่จำเป็นทั้งหมด

flutter pub get

รันแอปพลิเคชัน

flutter run

📁 โครงสร้างโปรเจกต์
mobileAppProject/
├── lib/                       # โค้ดหลักของ Flutter
│   ├── main.dart              # จุดเริ่มต้นของแอป
│   ├── screens/               # หน้าจอต่างๆ เช่น หน้าหลัก, จองห้อง
│   ├── models/                # โมเดลข้อมูล เช่น Room, Booking
│   ├── widgets/               # วิดเจ็ตที่นำกลับมาใช้ซ้ำ
│   └── services/              # บริการสำหรับ API หรือฐานข้อมูล
├── android/                   # โฟลเดอร์สำหรับ Android
├── ios/                       # โฟลเดอร์สำหรับ iOS
├── assets/                    # ไฟล์สื่อ เช่น รูปภาพ, ไอคอน
│   ├── screenshots/           # รูปหน้าจอสำหรับ README
│   └── gif/                   # GIF แสดงการใช้งาน
├── build/                     # โฟลเดอร์ที่สร้างโดย Flutter (ไม่ควรนำขึ้น Git)
├── pubspec.yaml               # รายการ dependencies และการตั้งค่าโปรเจกต์
├── .gitignore                 # ไฟล์ที่บอก Git ว่าจะไม่ติดตามไฟล์ใด
└── README.md                  # เอกสารโปรเจกต์ (ไฟล์นี้)
